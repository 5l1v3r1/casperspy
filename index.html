If i didn’t do it error notification “missing or corrupt codejock~1 component” will appear This is the content of Start.bat script:

@echo off echo :: --------------------------------------- echo :: Install and Register activex components echo :: --------------------------------------- pause for %%f in (*.ocx *.dll) do regsvr32 /s %%f echo :: ------- echo :: Succeed echo :: ------- echo :: -------------------- echo :: start CasperLauncher echo :: -------------------- pause start casperlauncher.exe
